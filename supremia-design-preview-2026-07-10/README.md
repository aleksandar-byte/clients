# Supremia Homepage Design Test

Static homepage prototype for Supremia Dentistry.

## Direction

- Main design guide: Grand Street Dental's editorial, personality-forward, hospitality-led feeling.
- Added structure from Forest Hills Dental: strong proof points, service architecture, review visibility, conversion flow.
- Added structure from SohoSmile: cleaner service grouping and neighborhood positioning.
- Uses real Supremia public facts and a small set of locally copied Supremia public site assets for review purposes.

## Notes

This is not deployed to WordPress. It is a visual/content prototype that can later be converted into ACF fields, template parts, CSS blocks, and a section registry.

The dental implants page follows a single treatment-service structure: local hero, candidacy, treatment explanation, planning process, implant options, doctor proof, cost/insurance context, related services, FAQ, and final CTA.
