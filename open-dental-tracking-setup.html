Open Dental Booking Tracking Setup

Please use these steps to set up the Open Dental booking link so completed online bookings can be tracked through a website thank-you page.

Thank-you page to use

Use the practice website domain with this page at the end:

https://www.YOURWEBSITE.com/thank-you-od/

Steps in Open Dental

  1. Open Open Dental.
  2. From the main menu, click eServices. Main Menu → eServices
  3. Under Web Scheduling, click Advanced. Web Scheduling → Advanced
  4. If there is more than one clinic/location, select the correct one from the list on the left.
  5. Find Scheduling URL.
  6. Right-click the Scheduling URL and choose Copy.
  7. Paste the copied link somewhere you can edit it.
  8. Add this to the very end of the link: &ReturnURL=https://www.YOURWEBSITE.com/thank-you-od/

Example

Original Open Dental link:

https://www.patientviewer.com/?RSID=34343338&CID=30&C=1251

Final link after adding the thank-you page:

https://www.patientviewer.com/?RSID=34343338&CID=30&C=1251&ReturnURL=https://www.YOURWEBSITE.com/thank-you-od/

Use &ReturnURL=, not ?ReturnURL=, because the Open Dental link already has a question mark in it.

Send us this

Please send us the final Open Dental link after the thank-you page has been added.

Final Open Dental booking link:
[paste final link here]

Do not send patient information. We only need the booking link.

Final check

  1. Open the final booking link.
  2. Complete a test booking only if your office is comfortable doing that.
  3. Confirm the browser ends on /thank-you-od/ after the booking is completed.
  4. Tell us when the test was completed so we can confirm tracking.

Open Dental reference: Web Sched Advanced manual. Open Dental documents the Scheduling URL and the &ReturnURL= option for redirecting patients after scheduling.
