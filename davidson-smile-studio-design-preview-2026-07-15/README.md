# The Smile Studio redesign page system

Open `template-library.html` to review every page type, or open `index.html` to start from the homepage.

## Included page types

- Homepage
- Services hub and single treatment
- Doctors directory and doctor profile
- About and smile gallery
- Contact, patient information, and location
- Blog index and article
- Flexible standard content page

`TEMPLATE-MAP.md` explains how each design maps to WordPress, ACF, Site Settings, custom post types, and native post content.

This is a local static design prototype based on public site content and assets. Form buttons open the visitor's email application; they are not connected to a live form system. No live website content has been changed.
